# React Capacitor 2026 - Project Rules

## Project Configuration
- **Project Name**: React Capacitor 2026
- **Port**: 3000 (configured in `src/theme.ts`)
- **Framework**: React 19 + TypeScript + Vite

## Architecture
- All project-specific configuration (name, title, colors, fonts, languages) should be stored in `src/theme.ts`
- Use variables from the theme file throughout the application
- Avoid hardcoding values that should be configurable

## Development
- Use TypeScript for all code
- Follow camelCase for filenames
- Use English-related, short, meaningful names for files
- Compatible with Chromium-based browsers
- Avoid using newest solutions that might not be compatible with browsers upgraded within a year

## Cloning Platform
This installation serves as the cloning platform for further child projects.
